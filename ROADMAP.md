# Franklyn — Roadmap & Upgrade List

## Paid Services to Add (when ready)

| What | Why | Estimated Cost |
|------|-----|----------------|
| ElevenLabs or OpenAI TTS | Replace browser robot voice with warm human voice for Franklyn | ~$0.015 per 1k characters |
| Supabase | Database + auth so user data persists across devices and multiple users can have accounts | Free tier generous, ~$25/mo at scale |
| Anthropic API | Already connected. Monitor usage as check-ins and chat grow | Pay per token |

### Voice Upgrade Path
- [ ] Phase 1: Browser SpeechRecognition (free, works now, Chrome best)
